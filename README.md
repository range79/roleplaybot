#  Discord Role Play Bot

Hii bro   
This is a role play bot for Discord.

I built it with **Spring Boot + Ollama**.  
It runs AI locally, no cloud, no pay, just vibes 

You need Docker to run this, otherwise you’ll suffer.

---

## ⚙️ What You Need

- Docker
- Discord bot token
- Some patience 

---

##  .env File (Important)

Create a `.env` file in the project folder and put this inside:

```env
DISCORD_USERNAME=
DISCORD_TOKEN=
DISCORD_OWNER=

POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=
```
